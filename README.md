
# 🍕 Pizza Sales Analysis Dashboard
This project is an end-to-end data analysis of a pizza sales dataset using **SQL Server** and **Power BI**. The goal was to understand sales performance, customer ordering patterns, and product trends by writing SQL queries for data analysis and visualizing the insights through an interactive Power BI dashboard.

## 📌 Project Overview
I started by analyzing the raw sales data using SQL to calculate business KPIs and answer common business questions. The processed data was then used to build a Power BI dashboard that provides a quick overview of sales performance and customer behavior.
This project helped me strengthen my SQL querying skills while learning how to present meaningful insights through data visualization.

## 🛠️ Tools Used
- SQL Server
- SQL
- Power BI
- Microsoft Excel (for initial data review)

## 📊 Dashboard Highlights

The dashboard includes:
- Total Revenue
- Average Order Value
- Total Orders
- Total Pizzas Sold
- Average Pizzas per Order
- Daily Order Trend
- Monthly Order Trend
- Sales by Pizza Category
- Sales by Pizza Size
- Top & Bottom Performing Pizzas by:
  - Revenue
  - Quantity Sold
  - Number of Orders

## 🔍 SQL Analysis
The SQL queries cover:
- Revenue calculations
- Order metrics
- Sales percentages
- Time-based trends
- Product performance analysis
- Ranking top and bottom-selling pizzas
These queries were used as the foundation for the Power BI visualizations.

## 📈 Key Insights
Some interesting findings from the analysis include:
- Certain pizza categories contributed significantly more to total revenue than others.
- Order volume varied noticeably across weekdays and months.
- A small number of pizzas generated the majority of sales, while some consistently underperformed.
- Larger pizza sizes contributed a substantial share of overall revenue.
